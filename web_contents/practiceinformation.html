Gap in the market

Why there is a need for DataSource.

The pharmaceutical industry is one of the most dynamic sectors of world economics. In 2015 the Pharmaceuticals and Biotechnology sector ranked first by overall research and development(R&D). Despite this investment, the sector is ranked 5th for net sales growth effectively illustrating the nature of the Pharmaceutical industry. The market is unstable and there is intensive competition between companies across geographies.

Project Proposal

DataSource - the concept.

The aim of this project is to automate the gathering, storage and reporting of open source drug approval data for the use of Regulatory Affairs teams in the Pharmaceutical industry for competitive intelligence analysis.

Development Choices and Method

DataSource influences and approaches.

DataSource is a niche solution targeting a user base and area that our competitors are not. The system is coded in Python as it is popular for data analyisis and web development, also it has a fast development speed. The version control was done through GitHub.

DataSource is using a combination of web scraping and text classification to get drug approval data from the FDA website and assign a therapy group to each drug!

Next Steps

Future plans for DataSource.

The next step for this project would be to scale it up. It currently handles all therapeutic areas, now the modules need to be added to handle multiple geographies by replicating the FDA PoC. The backbone has been created so with additional time modules handling other websites like the TGA could be added to build up a complete dataset of therapeutic and geographic drug approvals.

Competitors

Why choose DataSource, specifically?

Currently Splunk is used in several industries, including healthcare, for which the company provides case studies on how the solution is used. There are three healthcare case studies describing how Splunk has been applied to industry problems regarding data gathering and analytics.

SAS are a world leading data analytics solution employing data mining, statistical analysis, text analytics and optimization to allow users to access, prepare and model data to improve the return on analytics investment. SAS is also used in the healthcare industry to provide insights into data.
